┌─ 🎯 功能说明 ──────────────────────────────────────────────────────────────┐
│                                                                            │
│ Agent 可以做什么？                                                       │
│                                                                            │
│ ✅ 市场查询                                                              │
│    问: "市场怎么样？" / "宽度是多少？"                                  │
│    答: 市场态势 (Bull/Neutral/Bear) + 宽度得分 + 推荐仓位               │
│                                                                            │
│ ✅ 查看订单                                                              │
│    问: "有什么股票可以买？" / "今天有订单吗？"                           │
│    答: 今日交易信号 + 排名 + 权重 + 总投资比例                          │
│                                                                            │
│ ✅ 系统更新                                                              │
│    问: "重新分析一遍" / "生成最新订单"                                  │
│    答: 运行完整的市场分析流程，生成新订单                               │
│                                                                            │
│ ✅ 参数查询                                                              │
│    问: "参数是什么？" / "怎样调整？"                                    │
│    答: 所有系统参数 + 调整建议                                           │
│                                                                            │
│ ✅ 性能查询                                                              │
│    问: "历史表现怎么样？" / "回测效果？"                                │
│    答: 年化收益 + Sharpe + 最大回撤 + 成功率                             │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

┌─ ⚠️ 常见问题 ──────────────────────────────────────────────────────────────┐
│                                                                            │
│ Q1: 安装时出错了怎么办？                                               │
│ A: 检查网络连接，尝试:                                                  │
│    pip install langchain openai python-dotenv -i                         │
│    https://pypi.tuna.tsinghua.edu.cn/simple                             │
│                                                                            │
│                                                                            │
│ Q2: API Key 无效怎么办？                                                │
│ A: 1. 确认从 OpenAI 正确复制了 Key (以 sk- 开头)                        │
│    2. 确认 .env 文件格式正确 (没有多余空格)                             │
│    3. 在 https://platform.openai.com/account/api-keys 测试一下         │
│                                                                            │
│                                                                            │
│ Q3: Agent 说"无法生成订单"怎么办？                                      │
│ A: 可能原因:                                                             │
│    1. 还未运行过系统 (试试 "运行系统更新")                              │
│    2. 市场态势为 Bear，系统拒绝入场 (这是正常的，风险管理机制)         │
│    3. 所有信号都被过滤掉了 (参数需要调整)                               │
│                                                                            │
│                                                                            │
│ Q4: 成本是多少？                                                        │
│ A: GPT-3.5-turbo (已配置):                                              │
│    • 每次查询: $0.001 - $0.005                                          │
│    • 每月 100 次: $0.5 - $2.5                                           │
│    • 推荐首充: $10 USD (可用 2-3 年!)                                   │
│                                                                            │
│                                                                            │
│ Q5: 能自动下单吗？                                                      │
│ A: 目前只生成建议订单，需要手工在交易软件中执行。                       │
│    后续计划集成交易 API (CTP/TT) 实现自动下单。                        │
